# 🚜 Farmer Assistance Web Application

## 📌 Overview

A smart web-based platform designed to empower farmers with real-time insights, automation support, and community collaboration. The system integrates weather updates, motor control assistance, AI chatbot guidance, and government scheme awareness to improve agricultural productivity and decision-making.

---

## 🚀 Key Features

* 🌦️ **Real-time Weather Updates** – Get weather data based on city input
* 🚰 **Smart Motor Control System** – Assist in managing irrigation efficiently
* 🤖 **AI Chatbot Support** – Provides instant guidance for farming-related queries
* 📊 **Market Price Insights** – Helps farmers track crop prices
* 📢 **Government Schemes Section** – Direct access to useful agricultural schemes
* 💬 **Farmers Community Chat** – Share knowledge and suggestions
* 📱 **User-Friendly Interface** – Simple and accessible design

---

## 🛠️ Tech Stack
